Junction Annotation & Network Utility System

Model-Based Electric Diagram and TID Generation

Welcome to JANUS

If you want to create a new project, click on the "Create Project" option in the menu, Then upload the zip file containing the electrical diagram. It must be the html exported option from Visio

If you want to work on an existing project, click on the "Select Project" option in the menu

Once complete with the project, click on the "Export" option in the menu and it will generate the TID file and automatically download it as .docx file

If you want to logout, click on the "Logout" option in the menu
